# 문자열 입력

### cin 으로 받을때

1. 입력되는 문자 단위로 받기

   - `char` 형태로 받기

   ```c++
   char word[10];
   cin >> word ; 
   ```

   >  sort함수를 사용시 호출 형태
   >
   > `sort(begin, end, less<char>());`

   >문자열 크기 알고자 할 때
   >
   >`sizeof(word);`

   - `string` 함수: 문자 형으로 받기

   ```c++
   string word;
   cin >> word;
   ```

   > sort함수를 사용시 호출 형태
   >
   > `sort(word.begin(), word.end(), less<char>());` 

   > 문자열 크기 알고자 할 때
   >
   > `word.size();`

2. 입력되는 문장(띄어쓰기 포함) 단위로 받기

   - `char` 형태로 받기

   ```c++
   char word[10];
   cin.getline(c, 10);
   ```

   

   - `string` 함수

   ```c++
   string word;
   getline(cin, s);
   ```



###  선언할때

1. `char` 형태 입력

   ```c++
   char word[4] = {'a','b','c','d'}
   char word[10] = {};  // 모든 원소가 0으로 초기화
   ```



2. `string` 함수

   ```C++
   string word = {'a','b','c','d'}
   ```

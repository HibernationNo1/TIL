# exit

exit(): 



#### **return과의 다른 점**

1. return은 뒤 문장을 실행하며 종료하지만, exit()함수는 바로 프로세스를 종료한다.

2. return은 이전 함수로 복귀하지만(메인함수에서는 종료),  exit()함수는 바로 프로세스를 종료한다.



#### exit() 의미

동작은 전부 동일하다. (프로세스를 종료) 

단지 사용자가 자기만의 리턴 값을 정해서 에러를 처리하는 것이다.

`exit(0)` : 문제 없음

`exit(1)` : 파일 에러

`exit(-1)` : 치명적 에러
